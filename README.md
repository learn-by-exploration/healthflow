# HealthFlow

Health & medical management microservice for Synclyf.

## Status

**Pending implementation** — scaffolding only.

## Planned Scope

- Health vitals tracking (blood pressure, blood sugar, weight)
- Medication management & reminders
- Doctor appointments & visit history
- Emergency health cards (allergies, blood group, insurance)
- Family member health profiles

## Tech Stack

- Node.js / Express
- PostgreSQL
- Docker

## Part of

[Synclyf](https://github.com/learn-by-exploration/synclyf) — included as a git submodule under `services/healthflow/`.
